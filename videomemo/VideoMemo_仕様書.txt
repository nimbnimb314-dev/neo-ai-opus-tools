================================================================
動画メモアプリ（VideoMemo）仕様書 v1.0
Null and One
================================================================

■ 概要
動画編集者向けのローカル動作Webアプリ。
Insta360等のアクションカム動画をアップロードし、Whisperで音声を文字起こし。
タイムライン上でメモを管理して動画編集作業を効率化する。
完全無料・オープンソース。

■ これが出力される
  00:01:23  「ここメモ、桜の木がきれい」
  00:03:45  「メモ、逆光がいい感じ」
  00:07:12  「チェック、この転倒シーン使える」
  ...
  → CSV / TXT / Premiere用マーカーXMLとしてエクスポート可能

================================================================
1. 技術スタック
================================================================

  バックエンド    : Python / FastAPI
  音声抽出        : ffmpeg
  文字起こし      : faster-whisper（ローカルWhisper）
  DB              : SQLite
  フロントエンド  : React + Vite
  動画プレイヤー  : Video.js
  スタイル        : Tailwind CSS

================================================================
2. 機能仕様
================================================================

【動画アップロード・処理】
  - 対応フォーマット: mp4, mov, insv（Insta360形式）
  - アップロード後にffmpegで音声を自動抽出
  - faster-whisperで文字起こし（日本語・英語対応）
  - 文字起こし結果にタイムスタンプ付与

【タイムライン表示】
  - 動画プレイヤーとタイムラインを同期表示
  - 文字起こしテキストをタイムライン上にオーバーレイ
  - キーワード（「メモ」等）を自動検出してマーカー表示
  - ユーザーが任意の時点に手動でチェックマークを追加可能

【テキスト出力】
  - 全文書き起こしをタイムスタンプ付きで表示・コピー可能
  - メモ地点のみ抽出したリストも別途表示
  - どちらもワンクリックでTXTダウンロード

【メモ管理】
  - チェックマーク地点にテキストメモを記入
  - メモ一覧をサイドパネルに表示
  - タイムスタンプとメモ内容をリストエクスポート

【キーワード検出】
  デフォルトキーワード（設定画面で変更可能）:
  - 「メモ」「memo」「mark」「チェック」
  検出時の挙動:
  - タイムライン上に自動マーカーを追加
  - 通知音（オプション）

================================================================
3. エクスポート形式
================================================================

  CSV（メモのみ）    : 時間・発言・メモ → Excel / Numbers
  TXT（全文）        : 全文字起こし＋タイムスタンプ
  TXT（メモのみ）    : メモ地点のみ抽出
  XML（Premiere）    : マーカーデータ → Adobe Premiere Pro
  EDL（予定）        : 編集決定リスト → DaVinci Resolve

================================================================
4. APIエンドポイント
================================================================

  POST  /upload              動画ファイルをアップロード・処理開始
  GET   /status/{id}         処理状況のポーリング
  GET   /transcript/{id}     文字起こし結果取得
  POST  /memo                メモを保存
  GET   /memo/{video_id}     メモ一覧取得
  GET   /export/{id}         エクスポートファイル生成・ダウンロード

================================================================
5. UI構成
================================================================

  ヘッダー              : アプリ名 + 動画選択ボタン
  メインエリア（左）    : 動画プレイヤー + タイムライン
  サイドパネル（右）    : 文字起こし一覧 + メモ一覧
  フッター              : エクスポートボタン + キーワード設定

  タイムラインUI:
  - 再生バーの下に文字起こしテキストをスクロール表示
  - 自動検出マーカー: 黄色アイコン
  - 手動メモマーカー: 青色アイコン
  - マーカークリックで動画がその時点にジャンプ

================================================================
6. 開発ロードマップ
================================================================

  Phase 1  バックエンド基盤    FastAPI + ffmpeg + faster-whisper の動作確認
  Phase 2  基本UI              動画プレイヤー + タイムライン表示
  Phase 3  メモ機能            チェックマーク追加・テキストメモ保存
  Phase 4  キーワード検出      自動マーカー + 設定画面
  Phase 5  エクスポート        CSV / TXT / Premiere XML出力

================================================================
7. 制約・前提条件
================================================================

  - 完全オフライン動作（クラウドAPI不使用）
  - ユーザーはPython環境を持つ（インストールガイド同梱）
  - 初期対応言語: 日本語・英語
  - Insta360動画のinsv形式はffmpegで対応
  - GPUなし環境でもCPUモードで動作（処理は遅くなる）

================================================================
8. ポートフォリオとしての訴求点
================================================================

  - ローカルWhisperを使ったオフライン音声処理
  - FastAPI + React のフルスタック構成
  - 動画編集ワークフローへの実用的な価値
  - Adobe Premiere連携（プロ向け機能）
  - Null and One ブランドでGitHub / YouTubeに公開
